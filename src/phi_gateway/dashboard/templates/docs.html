Quick Start

Phi AI Gateway provides all primitives AI agents need in one API.

1. Get an API key

curl -X POST https://api.phiconsulting.biz.id/v1/keys \
  -H "Content-Type: application/json" \
  -d '{"name": "dev", "tier": "free"}'

2. Chat with any LLM

curl https://api.phiconsulting.biz.id/v1/chat/completions \
  -H "Authorization: Bearer <key>" \
  -H "Content-Type: application/json" \
  -d '{"model": "groq/llama-3.3-70b", "messages": [{"role": "user", "content": "Hello!"}]}'

3. Register a tool

curl -X POST https://api.phiconsulting.biz.id/v1/tools \
  -H "Authorization: Bearer <key>" \
  -H "Content-Type: application/json" \
  -d '{"name": "search", "description": "Web search", "json_schema": {"type": "object", "properties": {}}, "endpoint": "https://..."}'

4. Use MCP

curl -X POST https://api.phiconsulting.biz.id/mcp \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc": "2.0", "method": "tools/list", "params": {}, "id": "1"}'
