One API for AI agents

LLM Proxy, Tool Registry, Knowledge Base, and Agent Memory. Self-host on your own server or use our managed cloud. Built for developers who build AI agents.

📡

LLM Proxy

OpenAI-compatible endpoint. Routes to 10+ models across OpenAI, Anthropic, Groq, and OpenRouter. Cost tracking per request.

🛠

Tool Registry

Register tools with JSON Schema contracts. Agents discover and call them via REST or MCP (JSON-RPC 2.0).

📚

Knowledge Base

Ingest documents, chunk by paragraph, generate embeddings, search with cosine similarity. Keyword fallback built in.

🧠

Agent Memory

Persistent conversation storage with session management, pagination, and automatic context window trimming.

How it works

1

Deploy

One docker compose command. Runs on any VPS with 4GB RAM.

2

Connect

Point your LLM app or AI agent to the API. OpenAI-compatible format.

3

Build

Register tools, ingest knowledge, store conversations. All through one API.
